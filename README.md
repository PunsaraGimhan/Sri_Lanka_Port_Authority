# Sri Lanka Port Authority

A web-based management system for Sri Lanka Port Authority.

## Tech Stack

- **Frontend:** React.js
- **Backend:** PHP
- **Database:** MySQL

## Project Structure

- `frontend/` - React.js Frontend
- `backend/` - PHP Backend API
- `database/` - MySQL Database files

## Setup Instructions

### Database
1. Import `database/127_0_0_1.sql` to MySQL
2. Configure database credentials in `backend/config/.env`

### Backend
1. Move `backend/` folder to XAMPP `htdocs`
2. Start Apache and MySQL in XAMPP

### Frontend
```bash
cd frontend
npm install
npm start
```
